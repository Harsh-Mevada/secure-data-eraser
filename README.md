
# Secure Data Eraser Tool

A Python-based GUI application to securely erase files using advanced data wiping techniques.

## Features
- Secure file deletion
- Multiple wiping methods:
  - Quick Wipe
  - DOD 5220.22-M
  - Gutmann (35 Passes)
- File protection (blocks system files)
- Progress tracking UI

## Tech Stack
- Python
- Tkinter (GUI)

## How to Run
```bash
python data_wipe_gui.py
